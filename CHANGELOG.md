# Changelog

All notable changes to this project will be documented in this file.

## [1.0.9]
- Added `TagFormField` for seamless form validation and saving.
- Added floating auto-complete suggestions dropdown using overlay targets.
- Added keyboard navigation: Arrow Left/Right to highlight tags, Backspace once to highlight/twice to delete, and Delete key to remove highlighted tags.
- Added support for custom `inputFormatters` and `maxTagLength` character limits on tags.
- Updated SDK constraints and resolved all deprecation warnings.

- Initial release preparations
- Added customizable tag input widget with multiple styles and layouts
- Added animations support and tag validation
